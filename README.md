# Institute of Digital Risk — Landing Page

A modern, responsive landing page for the **Institute of Digital Risk (IDR)** designed to showcase training programs, advisory services, and digital risk innovation.
The website focuses on **clean UI design, responsive layout, and smooth user experience** using pure **HTML, CSS, and JavaScript**.

---

## Project Overview

This project is a professional institutional landing page built to represent a digital risk academy that combines **academic insight with industry practice**.
The design highlights the institute’s core pillars:
* Training & Bootcamps
* Innovation & Research
* Advisory Services
* Industry Deployment Pipeline
The website features modern UI elements, smooth animations, and a structured layout suitable for academic or corporate institutions.

---

## ✨ Features

* 📱 **Fully Responsive Design**
  * Works seamlessly on desktop, tablet, and mobile devices.

* 🎨 **Modern UI / UX**
  * Clean typography using Google Fonts
  * Dark theme with accent color design

* ⚡ **Interactive Elements**
  * Animated service cards
  * Scroll-based animations
  * Interactive contact form

* 📊 **Structured Sections**
  * Hero Section
  * About the Institute
  * Training & Deployment Pipeline
  * Service Offerings
  * Community / Audience
  * Contact Form

* 🍔 **Responsive Navigation Bar**
  * Collapsible mobile menu with hamburger button

---

## 🛠️ Technologies Used

* **HTML5** – Website structure

## Description of Logo
The isometric cube was chosen as the core icon because it simultaneously communicates **structure, layering, and resilience** — three qualities central to digital risk management. The three visible faces of the cube represent IDR's core pillars: academic knowledge (top/lit face in orange), industry practice (left/dark face), and deployment (right/mid-tone face), with a central dot symbolising the practitioner at the intersection
